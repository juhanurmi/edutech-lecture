Tietokannat

  • Organisoitu kokoelma dataa
  • DBMS (Data Base Management System) on ohjelmisto, joka huolehtii tietokannan luonnista, ylläpidosta ja datan käytöstä
  • Eräitä DBMS-järjestelmiä ovat MySQL, PostgreSQL ja MongoDB

Tietokannat

  • Relaatiotietokannat ja ei-relaatiotietokannat (SQL- ja NoSQL-tietokannat)
  • Structured Query Language (SQL) on kyselykieli relaatiotietokannoille
  • NoSQL (Not only SQL)

Tietokannat - SQL

  • Relaatiotietokannoilla on skeema
  • Yleensä takaavat ASID-ominaisuudet (Atomicity, Consistancy, Isolation, Durability)
  • Olemassaolevan tietokannan skeeman muuttaminen saattaa olla hankalaa

Tietokannat - NoSQL

  • NoSQL-tietokannoilla ei yleensä ole pysyvää skeemaa
  • Eivät takaa kaikkia ASID-ominaisuuksia
  • SQL-kieltä joustavammat kyselykielet
  • Suurien datamäärien hallinta

Tietokannat - NoSQL

  • Nopeita ja skaalautuvia
  • Hajautettavia
  • Jos tarvitaan ASID-ominaisuuksia, niin NoSQL ei tule kysymykseen!
  • Ketteriä, koska skeemaa ei ole
  • Erityisesti käytössä big datan ja web-sovellusten tietokantana

Tietokannat - NoSQL

NoSQL-ohjelmistoja

  • Amazon SimpleDB
  • MongoDB
  • Cloudata
  • Redis

  • Yksinkertainen JSON-tietokanta
  • Avain-arvo-parit
  • Suosituimpia NoSQL-tietokantoja
						 
luennoitsija = {"name" : "Juha", "age" : 25};
db.users.save(luennoitsija);

Verrattuna SQL-tietokantoihin

  • Dynaaminen skeema
  • Skaalautuva
  • Ketterä
  • Voidaan jakaa osiin mielivaltaisesti

MongoDB:n ketteryys

  • Ei tarvetta työläille migraatioille
  • Ei relaatioita
  • Yhteensopivuus esim. Djangon kanssa
  • Helppo rakentaa RESTful JSON API -ratkaisuja
  • Nopea ohjelmistokehitys ja tarve muuttuvalle skeemalle

MongoDB:tä ei sovi kaikkialle

  • Ei takaa ACID-ominaisuuksia
  • Jos skeema tiedetään tarkkaan ja se ei muutu
  • Haetaan äärimmäistä vakautta

Lopuksi

  • Relaatiotietokannat kankeita ketterässä kehityksessä
  • Skeemaa täytyy muuttaa usein
  • SQL-kieli on hankalaa lukea
  • NoSQL on joustava, mutta ei sovellu kaikkeen
  • MongoDB on eräs suosittu NoSQL-tietokanta
  • Siinä ohjelmointiläheinen kyselykieli
