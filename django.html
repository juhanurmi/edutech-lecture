Python

  • Python on yleiskäyttöinen ohjelmointikieli
  • Korkean tason kieli
  • Helppo lukuinen ja nopea oppia
  • Suuri määrä valmiita kirjastoja
  • Useita eri versioita kielestä

pip

  • Python-kielen kirjastojen asennusohjelma
  • Toimii kuten Linuxin apt tai yum
  • pip install some-package-name
  • Hallitsee riippuvuudet
  • pip install -r requirements.txt

virtualenv

  • Eristetty pakettien asenteluympäristö
  • Voidaan rakentaa erilaisia ympäristöjä samalle koneelle
  • Voi asentaa samasta kirjastosta eri versioita

Esimerkki

Ohjeet

						 
sudo apt-get install python
sudo apt-get install python-setuptools
sudo apt-get install python-pip
sudo apt-get install python-virtualenv
pip install virtualenv
mkdir suosio
cd suosio
pwd
/home/juha/suosio
virtualenv venv
source venv/bin/activate
pip freeze > requirements.txt

Django

  • Korkean abstraktiotason web-soveluskehys
  • Python-pohjainen
  • Ketterään web-kehitykseen
  • Suorituskykyinen
  • MVC-mallinen sovelluskehys
  • ORM (object-relational mapper) piilottaa tietokannan

Djangon asennus mongodb-enginen kanssa

					  	 
pip install django
pip install django-mongodb-engine
pip install django-toolbox
pip install mongoengine

Django-sovelluksen luonti

                                                 
django-admin.py startproject suosio_django .
python manage.py startapp
cd suosio_django/
mkdir templates
mkdir static
